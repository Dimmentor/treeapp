### 2) –ß—Ç–æ —Ç–∞–∫–æ–µ OuterRef
OuterRef - —ç—Ç–æ –∫–ª–∞—Å—Å, –∫–æ—Ç–æ—Ä—ã–π –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ –ø–æ–¥–∑–∞–ø—Ä–æ—Å–æ–≤ (subqueries) –¥–ª—è —Å—Å—ã–ª–∫–∏ –Ω–∞ –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ –≤–Ω–µ—à–Ω–µ–≥–æ –∑–∞–ø—Ä–æ—Å–∞. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏—é –∏–ª–∏ –¥—Ä—É–≥–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤ –ø–æ–¥–∑–∞–ø—Ä–æ—Å–µ, –æ—Å–Ω–æ–≤—ã–≤–∞—è—Å—å –Ω–∞ –¥–∞–Ω–Ω—ã—Ö –∏–∑ –æ—Å–Ω–æ–≤–Ω–æ–≥–æ –∑–∞–ø—Ä–æ—Å–∞. 

OuterRef –∏–≥—Ä–∞–µ—Ç –∫–ª—é—á–µ–≤—É—é —Ä–æ–ª—å –≤ —Ä–∞–±–æ—Ç–µ —Å –ø–æ–¥–∑–∞–ø—Ä–æ—Å–∞–º–∏, –æ—Å–æ–±–µ–Ω–Ω–æ –∫–æ–≥–¥–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —Å–æ–æ—Ç–Ω–µ—Å—Ç–∏ –¥–∞–Ω–Ω—ã–µ –∏–∑ –≤–Ω–µ—à–Ω–µ–≥–æ –∑–∞–ø—Ä–æ—Å–∞ —Å –¥–∞–Ω–Ω—ã–º–∏ –≤–Ω—É—Ç—Ä–∏ –ø–æ–¥–∑–∞–ø—Ä–æ—Å–∞. 

–ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —É –≤–∞—Å –µ—Å—Ç—å –º–æ–¥–µ–ª–∏ Author –∏ Book, –∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–æ–ª—É—á–∏—Ç—å –≤—Å–µ—Ö –∞–≤—Ç–æ—Ä–æ–≤, —É –∫–æ—Ç–æ—Ä—ã—Ö –µ—Å—Ç—å —Ö–æ—Ç—è –±—ã –æ–¥–Ω–∞ –∫–Ω–∏–≥–∞ —Å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º –Ω–∞–∑–≤–∞–Ω–∏–µ–º. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ –≤—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å OuterRef –¥–ª—è —Å—Å—ã–ª–∫–∏ –Ω–∞ primary key (pk) –∞–≤—Ç–æ—Ä–∞ –≤–Ω—É—Ç—Ä–∏ –ø–æ–¥–∑–∞–ø—Ä–æ—Å–∞: 

```sh
from django.db.models import OuterRef, Subquery, Count

books_with_title = Book.objects.filter(
    author=OuterRef('pk'),  # –°—Å—ã–ª–∫–∞ –Ω–∞ pk –∞–≤—Ç–æ—Ä–∞ –∏–∑ –≤–Ω–µ—à–Ω–µ–≥–æ –∑–∞–ø—Ä–æ—Å–∞
    title="My Awesome Book"

authors_with_book = Author.objects.annotate(
    num_books=Subquery(books_with_title.values('pk').annotate(count=Count('*')).values('count'))
).filter(num_books__gt=0)
```

from collections import deque

def bfs(start):
    queue = deque([start])
    visited = set([start])
    while queue:
        node = queue.popleft()
        for neighbor in node.neighbors:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
